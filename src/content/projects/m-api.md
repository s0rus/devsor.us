Discord bot for private discord server that handles commands and aggregates data built with:

- Bun
- Hono
- Prisma
- Supabase
- Discord.js
- Twurple
- Bunch of utility like zod, chalk, dayjs, env.t3

Hosted on private VPS using docker-compose
